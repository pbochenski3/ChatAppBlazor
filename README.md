ChatApp 🚀
Real-time chat application built with .NET 10, featuring a Blazor WebAssembly frontend and a SignalR backend.

🌟 Key Features
Real-time Messaging: Instant message delivery using SignalR Hubs.

Dynamic Sidebar: Real-time updates for last messages and user aliases across all participants.

Security: Integrated message encryption using the Decorator Pattern in the repository layer.

State Management: Centralized state services for seamless UI updates without unnecessary page reloads.

Clean Architecture: Separated layers for API, Application, Domain, and Infrastructure.

MediatR Integration: Decoupled communication using the Mediator pattern for commands, queries, and notifications.
